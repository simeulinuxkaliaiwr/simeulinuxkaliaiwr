### 🔧 Current Setup & Philosophy
```

GRUB_CMDLINE_LINUX_DEFAULT="loglevel=3 quiet \
lsm=lockdown,yama,apparmor,bpf \
random.trust_cpu=0 \
slab_nomerge init_on_alloc=1 init_on_free=1 page_poison=1 \
```
